### Manually Testing using REPL

To start in repl mode run `yarn repl`. This will start an interactive repl, in which you can invoke and test the OneSpanSign class

**Example**

```sh
// Step 1: Start repl
yarn repl

// Step 2: Test code
> console.log(OneSpanSign)
```
